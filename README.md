# Esquema de Desarrollo

Este modelo propone utilizar **ADO**, como tecnología para el acceso de datos con DataSet desconectados. La manera de ejecutar consultas y procedimientos almacenados es con el sqlDataAdapter y sus comandos. 

## Instalación
Para poder ejecutar el ejemplo se deben tener en cuenta los siguientes requisitos:

*  Visual Studio 2010 -2015
*  DevExpress v.15.2.7
*  Json.NET 9.0.1

	```c#
	PM> Install-Package Newtonsoft.Json -Version 9.0.1
	```
